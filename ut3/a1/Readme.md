#### ***Introducción***. <a name="id1"></a>

El proyecto Eclipse fue iniciado por **IBM** en noviembre de 2001 y, posteriormente, en 2004, se estableció la **Fundación Eclipse** como una organización independiente. Eclipse es conocido por su robusto soporte para **Java**  , pero es una herramienta **multilenguaje** que soporta una amplia gama de lenguajes y tecnologías, incluyendo C/C++, PHP, Python, JavaScript, y herramientas para desarrollo web, bases de datos, y servidores de aplicaciones.
